# Ideal Customer Profile

## 一级目标客户

Synapse v1 的理想客户是正在建设企业内 AI 系统、且必须自己控制执行边界的团队。

优先级最高的 ICP：

- 内部 AI 代码助手团队
- DevInfra / PR 自动化团队
- 高合规行业的 AI 平台团队

## 组织特征

这些团队通常具备以下共同特征：

- 已经让模型参与代码、脚本或数据处理流程
- 无法接受执行过程出域或完全托管在第三方云
- 安全、平台、审计团队对执行边界有明确要求
- 愿意以私有化部署换取控制力和合规性
- 需要在短时间内验证一个真实 PoC

## 关键买点

- 不出域
- 可审计
- 不失控
- 私有化部署
- 对现有企业 AI 平台可嵌入

## 不优先客户

以下客户暂不作为 v1 主攻方向：

- 个人开发者
- 只追求最低单次执行成本的 API 用户
- 需要浏览器或桌面自动化的团队
- 追求完整在线 IDE 平台的客户
