# ICP Target List Template

## 字段模板

每个潜在客户至少记录以下信息：

| 字段 | 说明 |
| --- | --- |
| Company | 公司或 BU 名称 |
| Team | 目标团队名称 |
| Contact | 关键联系人 |
| Role | 联系人角色 |
| Use Case | 当前 AI 执行场景 |
| Current Stack | 当前使用的模型、平台、执行方式 |
| Compliance Need | 是否有内网、审计、数据边界要求 |
| Trigger Event | 当前推动项目的触发因素 |
| Pain Point | 现有方案的核心问题 |
| PoC Fit | 是否适合 Synapse v1 PoC |
| Next Step | 下一步动作 |
| Status | `to_reach`, `contacted`, `qualified`, `poc`, `closed_lost`, `closed_won` |

## 建议评分

每个线索建议按 1 到 5 分评估：

- 执行需求强度
- 合规要求强度
- 私有化偏好
- 当前接入紧迫度
- 团队执行能力

总分越高，越优先进入设计合作客户名单。
