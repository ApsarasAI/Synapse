# Not In v1

## 本阶段明确不做

以下方向不属于 Synapse v1：

- 多语言运行时扩展
- 浏览器自动化、Desktop、Computer Use
- 在线 IDE、云工作站、交互式开发环境
- 模板市场、快照平台、复杂控制面
- 面向个人开发者的低价按量模式
- 公有云托管版
- 以“替代 E2B”作为对外主叙事

## 处理原则

如果新增需求落入以上范围，默认不进入当前版本，除非经过首版边界评审并能证明：

- 直接支撑企业 PoC 成交
- 不破坏 Python-only 与私有化优先策略
- 不影响 API v1、SDK、文档、demo、release gate 的交付顺序
